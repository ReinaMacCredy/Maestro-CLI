## index (memory key)

```markdown
# Maestro Context Index

> Auto-generated by `maestro:setup`. Update this file when adding new context documents.

## Context Files (Global Memory)

| Key | Purpose |
|-----|---------|
| product | Product definition -- purpose, users, features |
| tech-stack | Technology stack -- languages, frameworks, tools |
| guidelines | Coding guidelines -- principles, conventions, NFRs |
| product-guidelines | Product guidelines -- voice, tone, UX, branding |
| workflow | Workflow configuration -- methodology, commits, coverage |
{if style guides: | code_styleguides/ | Code style guides for detected languages |}

## Features

Use `maestro feature-list` (CLI) or `maestro_feature_list` (MCP) to view all features.

## Usage

Skills resolve context by reading global memory first, then loading the relevant entries.
```
